# Sweet Crumbs Bakery (HTML + CSS)

A clean, modern, and fully responsive single-page website for a bakery. Built with only HTML and CSS.

## Structure
- `index.html`: Page layout (hero, menu, about, gallery, testimonials, contact, footer)
- `styles.css`: Responsive styling, elegant typography, and warm color palette

## Preview locally
Open `index.html` in your browser, or run a simple static server:

```bash
cd /workspace/bakery-site
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Customize
- Text: Update copy in `index.html`.
- Colors: Adjust CSS variables in `styles.css` under `:root`.
- Images: Replace the Unsplash image URLs with your own assets.
- Menu: Duplicate a `.card` block to add more items.

## Notes
- No JavaScript required. Everything is HTML/CSS.
- Tested for responsive behavior down to small phones.
